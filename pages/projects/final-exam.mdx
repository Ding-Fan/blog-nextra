### 23

問1
幕末の志士である佐久間象山は、近代日本における（ ）という考え方を、「東洋道徳、西洋芸術」と表現していた。

1. 士魂商才
2. 和魂漢才
3. 和魂洋才
4. 和洋折衷

正解：3. 和魂洋才

解説：
「和魂洋才」は、日本の精神や伝統（和魂）を基盤に、西洋の技術や知識（洋才）を積極的に取り入れるという思想を指します。佐久間象山は「東洋道徳、西洋芸術」という表現で、この考えを提唱しました。 

問2
情報技術（IT）は（ ）の勢いで進化し続けている。

1. 花鳥風月
2. 風光明媚
3. 日進月歩
4. 風林火山

正解：3. 日進月歩

解説：
「日進月歩」は、日ごとに進み、月ごとに進歩するという意味で、技術や事柄が急速に進化するさまを表します。情報技術の発展を表現する際に適切です。 

問3
（ウ）会社中がそのウワサで（ ）だ。

1. しめきり
2. かしきり
3. いいきり
4. たべきり
5. もちきり

正解：5. もちきり

解説：
「もちきり」は、ある話題がずっと続いている状態を表します。「会社中がそのウワサでもちきりだ」は、「そのウワサの話題で持ちきりだ」という意味です。 

### 24

（ア） 試験勉強をしなければならないので、クリスマス（ ）ではない。
	1.	だけ
	2.	ばかり
	3.	どころ
	4.	など

正解：3. どころ

解説： 「～どころではない」は、ある状況や理由により、他のことをする余裕がないことを表します。この場合、試験勉強が忙しくて、クリスマスを楽しむ余裕がないことを意味します。

（イ） 新型コロナに感染したため、学校を休ま（ ）。
	1.	ざるをいる
	2.	ざるをえる
	3.	ざるをいない
	4.	ざるをえない

正解：4. ざるをえない

解説： 「～ざるを得ない」は、「～しなければならない」という意味で、避けられない状況を表します。この場合、新型コロナに感染したため、学校を休まなければならないことを意味します。

（ウ） 言う（ ）ありませんが、正月にも勉強する必要があります。
	1.	ことも
	2.	までも
	3.	ほども
	4.	ほどでも

正解：2. までも

解説： 「言うまでもありませんが」は、「あれこれ言う必要のないほどわかりきったことである。もちろんである。」という意味の表現です。  ￼この場合、「正月にも勉強する必要がある」ということが明らかであることを強調しています。

（エ） 理容室に行かないまでも、せめて（ ）の髪は整えてから行った方が良い。
	1.	つるつる
	2.	つやつや
	3.	ぼうぼう
	4.	ぼさぼさ
	5.	ふさふさ

正解：4. ぼさぼさ

解説： 「ぼさぼさ」は、髪の毛などが整っておらず、乱れている様子を表します。この場合、理容室に行かなくても、せめて乱れた髪は整えてから出かけた方が良いという意味です。

### 25

問1：日本の労働契約についての説明として正しいものはどれですか。
	1.	雇用契約でも請負契約でも労働者の保護に差はない。
	2.	委任契約は仕事の完成を約束する契約である。
	3.	請負契約は何らかの行為を依頼する契約である。
	4.	正社員もアルバイトも雇用契約に分類される。
	5.	会社の社長は雇用契約によって会社に雇われている。

正解：4. 正社員もアルバイトも雇用契約に分類される。

解説： 正社員もアルバイトも、労働者として企業と雇用契約を結びます。雇用契約とは、労働者が企業に労働を提供し、その対価として賃金を受け取る契約を指します。雇用形態の違いはありますが、いずれも雇用契約に分類されます。

問2：日本の雇用契約についての説明として誤っているものはどれですか。
	1.	アルバイトとは、短時間または短期間だけ働く契約である。
	2.	派遣社員とは、派遣会社から紹介された他の会社で働く契約である。
	3.	請負契約は何らかの行為を依頼する契約である。
	4.	契約社員とは、契約期間の定めがない契約である。
	5.	正社員とは、原則として定年まで働くことができる契約である。

正解：4. 契約社員とは、契約期間の定めがない契約である。

解説： 契約社員とは、企業と一定の契約期間を定めて雇用契約を結ぶ労働者を指します。したがって、契約期間の定めがないという説明は誤りです。

### 26

（ア） 私の一存では（ ）ので社に持ち帰らせていただきます。
	1.	決めれる
	2.	決められる
	3.	決めれない
	4.	決められない

正解：4. 決められない

解説： 「決める」の可能形は「決められる」です。否定形は「決められない」となります。「決めれない」は口語で使われることがありますが、正式な表現ではありません。したがって、「私の一存では決められないので、社に持ち帰らせていただきます。」が正しい表現です。

（イ） 今回のテストは難しくて（ ）漢字が多かった。
	1.	読めない
	2.	読めれない
	3.	読められない
	4.	読めることができない

正解：1. 読めない

解説： 「読む」の可能形は「読める」であり、否定形は「読めない」です。「読めれない」や「読められない」は誤った表現です。また、「読めることができない」は冗長な表現となります。したがって、「今回のテストは難しくて読めない漢字が多かった。」が正しい表現です。

（ウ） 申し訳ありません。本日は体調が悪いため（ ）いただきます。
	1.	休ませて
	2.	休まさせて
	3.	休ませないで
	4.	休まさせないで

正解：1. 休ませて

解説： 「休ませていただきます」は、自分が休むことを相手に丁寧に伝える表現です。「休まさせていただきます」は誤った敬語表現であり、適切ではありません。したがって、「申し訳ありません。本日は体調が悪いため休ませていただきます。」が正しい表現です。

### 27

（ア） 試験前の教室は、（ ）と静まり返っていた。
	1.	しん
	2.	しんしん
	3.	つん
	4.	つんつん

正解：1. しん

解説： 「しんと」は、周囲が非常に静まり返っている様子を表す擬態語です。試験前の緊張感ある教室の静けさを強調しています。

（イ） 請求書を添付いたしましたので（ ）ください。
	1.	ご笑納
	2.	ご検収
	3.	ご査収
	4.	ご賞味
	5.	ご堪能

正解：3. ご査収

解説： 「ご査収ください」は、送付した書類や資料を確認して受け取ってほしいと依頼するビジネス表現です。請求書などの重要な書類を送る際に適切な表現です。

（ウ） テストの点数が良かったからといって、気を抜くと（ ）をすくわれるよ。
	1.	手
	2.	手元
	3.	足
	4.	足元
	5.	口
	6.	口元

正解：4. 足元

解説： 「足元をすくわれる」は、油断しているときに相手から不意打ちを受けて失敗することを意味する慣用句です。テストの点数が良くても、気を抜くと予期せぬ失敗をする可能性があることを警告しています。

### 28

（ア） 上司が代わって以降、職場の（ ）がよくなりました。
	1.	風除け
	2.	風当たり
	3.	風向き
	4.	風通し
	5.	風光る

正解：4. 風通し

解説： 「風通しがよい」は、職場などで情報や意見の交換がスムーズで、開かれた雰囲気を指します。上司が代わったことで、職場のコミュニケーションが円滑になったことを表しています。

（イ） 疲れてきたかと思いますので、切りの良いところで一（ ）しましょう。
	1.	一喝
	2.	一筆
	3.	一服
	4.	一考
	5.	一任

正解：3. 一服

解説： 「一服する」は、休憩を取ることを意味します。疲れを感じたときに、区切りの良いところで休憩を提案する文脈に適しています。

（ウ） この度は私どもの（ ）でご迷惑をお掛けしました。
	1.	無作法
	2.	無遠慮
	3.	非常識
	4.	不躾
	5.	不手際

正解：5. 不手際

解説： 「不手際」は、物事の処理が適切でないこと、手際が悪いことを指します。自社のミスや手配の不備で相手に迷惑をかけた際の謝罪表現として適切です。

（エ） トラブルが生じたため、プロジェクトは（ ）。

解説： 「棚上げにする」は、問題や案件の処理を一時的に保留することを意味します。トラブルが発生したため、プロジェクトを一時中断する状況を表しています。

### 29

（ア） 新しいプロジェクトでリーダーを（ ）ことになった。
	1.	努める
	2.	勉める
	3.	務める
	4.	勤める

正解：3. 務める

解説： 「務める」は、役割や任務を引き受けて行うことを意味します。この場合、プロジェクトのリーダーという役割を引き受けることを表しています。一方、「勤める」は会社などに勤務すること、「努める」は努力することを指します。

（イ） お気に入りのレストランに行ったが、今日は（ ）いなかった。
	1.	相手
	2.	開いて
	3.	明いて
	4.	空いて

正解：4. 空いて

解説： 「空いている」は、場所や席などが使用中でないこと、つまり利用可能であることを意味します。この場合、レストランが営業していなかった、または満席で入れなかったことを表しています。

（ウ） 欠席した人の（ ）に私が説明することになった。
	1.	代わり
	2.	変わり
	3.	替わり
	4.	換わり

正解：1. 代わり

解説： 「代わり」は、他の人や物の代理として行動することを意味します。この場合、欠席した人の代理として説明を行うことを表しています。「変わり」は状態や性質の変化、「替わり」「換わり」は物や人を交換することを指します。

（エ） 俳句が好きとは、とても（ ）な趣味をお持ちですね。
	1.	公証
	2.	高尚
	3.	交渉
	4.	高床
	5.	校章
	6.	工廠

正解：2. 高尚

解説： 「高尚」は、知性や品位が高く、上品であることを意味します。俳句を趣味とすることは、文化的で品位の高い趣味と評価されています。

（オ） 日本には火山が多いため、（ ）や噴火などの災害が多い。
	1.	磁針
	2.	時針
	3.	自身
	4.	自信
	5.	地震
	6.	慈心

正解：5. 地震

解説： 「地震」は、地殻の変動によって地面が揺れる自然災害を指します。火山活動と地震は密接に関連しており、日本ではこれらの災害が多く発生します。
