# Programming Basics

### Decimal: 小数　しょうすう

- Any numeral value that consists of a decimal point is a decimal number.
- Example: `3.14`, `2.5`, `0.001`, `10.0`

#### In Java

- Decimal types are `float` and `double`.
- `float` means floating point
- `double` means it's twice of `float`

### Integer: 整数　せいすう

- A whole number that can be positive, negative, or zero, without any fractional or decimal parts.
- Example: `42`, `-15`, `0`, `1000`

#### In Java

- Integer types are `byte`, `short`, `int`, and `long`.
