# Flatpak CJK Font Issue

Telegram and Bitwarden (Flatpak) showing missing/broken Chinese characters, while VS Code and Brave display them fine.

**Problem**: Flatpak apps use bad default CJK fonts and can't access system fonts.

**Fix**:
```bash
# Check if they're Flatpak
flatpak list | grep -E "(telegram|bitwarden)"

# Grant font access
flatpak override --user --filesystem=xdg-data/fonts org.telegram.desktop
flatpak override --user --filesystem=/usr/share/fonts org.telegram.desktop

flatpak override --user --filesystem=xdg-data/fonts com.bitwarden.desktop  
flatpak override --user --filesystem=/usr/share/fonts com.bitwarden.desktop
```

Restart apps. Chinese characters should now display properly with better fonts.

The issue was Flatpak using poor default CJK fonts instead of system Noto CJK fonts - not just fontconfig.
