# MySQL

## Login to MySQL

**Input:**
```bash
sudo mysql -u root -p
# Press Enter because no password
```

You will see the MySQL prompt:
```
mysql>
```

## Show Databases

**Input:**
```sql
SHOW DATABASES;
```

**Output:**
```
+--------------------+
| Database           |
+--------------------+
| information_schema |
| mysql              |
| performance_schema |
| sys                |
+--------------------+
4 rows in set (0.02 sec)
```

## Create Database

**Input:**
```sql
CREATE DATABASE study;
```

**Output:**
```
Query OK, 1 row affected (0.04 sec)
```
