```json
{

}
```

## install mcp

install [chrome-devtools mcp](https://github.com/ChromeDevTools/chrome-devtools-mcp/?tab=readme-ov-file#chrome-devtools-mcp) at user(global) scope, the `~/.claude.json` will be modified.

```sh
claude mcp add chrome-devtools npx chrome-devtools-mcp@latest --scope user
```
