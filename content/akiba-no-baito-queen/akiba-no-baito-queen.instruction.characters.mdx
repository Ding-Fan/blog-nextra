# アキバのバイト女王 - キャラクター設定資料

## 概要

この資料は「アキバのバイト女王」に登場するキャラクターの詳細な設定情報を提供します。各キャラクターの外見、性格、話し方、役割について詳しく説明しています。

## 利用可能なキャラクター

### Character Component で使用できるキャラクター
- `tenchou` (店長) - Old otaku manager with cap and glasses
- `watashi` (私) - Young blonde girl learning Japanese
- `customer` (お客さん) - Store customer

### アバターシステム
- **自動アバター**: Character の `name` prop に基づいて自動的に読み込まれます
- **デフォルトパス**: 
  - `tenchou` → `/images/characters/tenchou-avatar.png`
  - `watashi` → `/images/characters/watashi-avatar.png`
  - `customer` → `/images/characters/customer-avatar.png`
- **カスタムオーバーライド**: `avatar="/custom/path.png"` でデフォルトを上書き可能
- **アバター無効化**: `noAvatar={true}` で絵文字のみ表示
- **フォールバック**: アバター画像の読み込みに失敗した場合、絵文字アバターが表示されます
- **ゲームライクな表示**: アバターがダイアログボックスと一緒に表示され、没入感のある体験を提供

## 詳細キャラクタープロファイル

### 店長 (Tenchou)
- **Physical Description**: 太ったオタクのおじさん
- **Appearance**: いつもキャップとサングラスをかけている
- **Technology**: サングラスはスマートメガネです、それを隠すため、いつもスマホモックアップを見ている
- **Personality Reference**: シュタインズ・ゲートの橋田至が年を取ったような感じ
- **Character Traits**: 少し面倒くさがりだが、心は優しい
- **Speech Style**: Casual, sometimes uses otaku slang, slightly impatient with newbies
- **Teaching Mode**: **重要** - 技術スキル（Excel、ソフトウェアなど）を教える時は、忍耐強く、親切で、ユーモラスな比喩を使って学習を助けます。普段のせっかちな性格とは全く違います。
- **Role**: 学習の旅で笑いとガイダンスの両方を提供

Anime/manga art style, warm lighting, character should look approachable but slightly mischievous about hiding his tech gadgets.

### お客さん (Customer)

- **Types**: オタク、サラリーマン、観光客など

## 感情表現システム

キャラクターコンポーネントで利用可能な感情:
- `normal` - デフォルト状態
- `angry` - 赤い影効果
- `happy` - 黄色い影で温かみを表現
- `confused` - グレーの影
- `tired` - 紫の影
- `excited` - 緑の影とバウンスアニメーション
- `nervous` - 不安を表現する水色の影
- `embarrassed` - 恥ずかしがる瞬間のピンクの影
- `thoughtful` - 熟考のための深い青の影
- `curious` - 好奇心旺盛な瞬間のオレンジの影

## キャラクター使用例

### 基本的な使用方法
```jsx
{/* 自動アバター - 推奨 */}
<Character name="tenchou" emotion="happy">
こんにちは！
</Character>

{/* カスタムアバターオーバーライド */}
<Character name="tenchou" avatar="/images/special-tenchou.png" emotion="happy">
特別な場面です！
</Character>

{/* アバターなし、絵文字のみ */}
<Character name="watashi" noAvatar={true} emotion="confused">
内心の独白...
</Character>
```

### アクションシーン
```jsx
<Character name="watashi" isAction>
nervously adjusts her blonde ponytail and the required cap
</Character>

<Character name="tenchou" isAction>
スマートメガネを調整しながらスマホモックアップを見る
</Character>
```

### 言語学習コンテキスト
```jsx
<Character name="watashi" emotion="confused">
あの...すみません、「ユニフォーム」って何ですか？
</Character>

<Character name="tenchou" emotion="tired">
制服だよ、制服。日本語まだまだだな...
</Character>
```
