# Fedora Installation Process

Complete step-by-step guide for installing Fedora Linux.

## Pre-Installation

### Creating Bootable USB
1. Download Fedora Workstation ISO 
2. Create bootable USB using Balena Etcher

### BIOS/UEFI Preparation
1. Enter BIOS/UEFI settings (usually F2, F12, or Del during boot, for thinkpad carbon x1, it might be press enter after the logo shows)
2. Disable Secure Boot if planning to use proprietary drivers
3. Enable UEFI boot mode

---

## Installation Process

### Step 1: Boot from USB
1. Insert USB and restart computer
2. Select USB from boot menu
3. Choose "Start Fedora-Workstation-Live"

### Step 2: Live Environment
1. Test hardware compatibility
2. Connect to WiFi if needed
3. Click "Install to Hard Drive"

### Step 3: Installation Configuration

#### Language & Keyboard
- Select your language and keyboard layout

#### Installation Destination
**Simple Setup:**
- Select "Automatic partitioning" for beginners

#### User Creation
1. Create user account
2. Set strong password

### Step 4: Complete Installation
1. Click "Begin Installation"
2. Wait for installation to complete (15-30 minutes)
3. Reboot and remove USB
